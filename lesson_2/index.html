Конспект лекції

Cинтаксис HTML: все що відкрито, має бути закритим!

Будова (анатомія) HTML-тегу

  1. Тег з зачиняючим елементом: <назва_тегу>Наповнення тегу (інші теги, контент) </назва_тегу>
  2. Тег без зачиняючого елементу: <назва_тегу>
  3. Тег з атрибутами (атрибутів може бути скільки завгодно):
    • <назва_тегу назва_атрибуту="значення_атрибуту" назва_атрибуту_2="значення_атрибуту_2"> Наповнення атрибуту (інші теги, контент) </назва_тегу>
    • <назва_тегу назва_атрибуту="значення_атрибуту">

Теги для будівництва - будівельні

асоціація з цими тегами виглядає так:

  1. DIV - будівельний тег БЕЗ конкретного призначення (з зачиняючим елементом)
  2. HEADER - будівельний тег для побудови верхньої частини сайту/блоку (з зачиняючим елементом)
  3. MAIN - будівельний тег для побудови головної частини сайту/блоку (має бути лише один на сторінці) (з зачиняючим елементом)
  4. FOOTER - будівельний тег для побудови нижньої частини сайту/блоку (з зачиняючим елементом)

Будівельні теги ніяк не змінюють контент

Теги для контенту - контентні

Контентні теги можуть змінювати контент:

  1. h1-h6 - Теги для заголовків (з зачиняючим елементом)
    • H1 - лише один на сторінці бо він головний
    • H2-H6 - можуть повторюватися
  2. P - тег параграфу (з зачиняючим елементом)
  3. MARK та SPAN - виділення слова(ів) (з зачиняючим елементом)
  4. BR - перериває рядок тексту (без зачиняючого елементу)
  5. UL+LI - Маркований список (з зачиняючим елементом)
  6. OL+LI - Нумерований список (з зачиняючим елементом)
  7. IMG - вивід зображення (без зачиняючого елементу)

    8. АТРИБУТИ

    • src - шлях до зображення (файлу)
    • alt - альтернативний текст
  8. A - гіпертекстове посилання (з зачиняючим елементом)

    9. АТРИБУТИ

    • href="адреса" - URL що має відкритися
    • target="_blank" - встановлює відкриття сторінки в новій вкладці
    • CLASS - додає клас(класи), пишемо латиницею, у нижньому регістрі, пробіл змінюємо на дефіс
    • TITLE - підказка, що відображається при наведенні на посилання
    • ID - унікальний ідентифікатор

Блочні теги

Це усі будівельні теги (div, header, main, footer та інші)

Контентні (h1-h6, p, ul, ol, li)

Ширина блочного тегу завжди займає всю ширину батьківського елементу

Висота формується контентом всередені

Блочний тег завжди починається з нового рядка

Рядкові теги

<span></span>, <mark></mark>, <a></a>, <br>

Ширина рядкового тегу займає ширину по ширині контенту всередені

Висота формується контентом всередені

Рядковий тег намагається розміститись в рядок разом з іншими рядковими тегами або текстом

Блочно-рядкові теги

<img>

Ширина блочно-рядкового тегу займає ширину по ширині контенту всередені

Висота формується контентом всередені

Блочно-рядковий тег намагається розміститись в рядок разом з іншими рядковими тегами або тексто
